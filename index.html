📖 User Instructions

Use this tool to filter and analyze data
from your LiveVox Call Detail Record (CDR) files.

Please follow these steps below:
  1. Convert to CSV: Open your Excel file, go to File > Save As and select CSV (Comma Delimited).
  2. Select File: Click "Choose File" and pick the .csv version.
  3. Enter Keywords: Paste your codes into the box. You can separate them with spaces, commas, or new lines.
  4. Run Search: Click "Run Search". The system will read the file and output rows sorted by your keyword list order.

    5. Note: (For faster searches)
    Keep only Session ID and Transaction ID in the CSV file.
    Ensure Transaction ID is formatted as a Number with NO decimal places.
